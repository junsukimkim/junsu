공모 알림

가족 체크 + 캘린더(.ics) + 미리알림(단축어) + 자동 가져오기

자동 가져오기

범위: 오늘 ~ 다음달 말 (이번달+다음달을 자동으로 합쳐서 보여줘요)

※ “공모주만” 필터는 상장사(유상증자/권리 등)를 최대한 제외하는 방식이에요. 최종은 공시로 확인!

가족 관리

가족 목록

기본 4명이 자동 생성돼요. 필요하면 이름만 바꿔도 OK.

캘린더(.ics) 내보내기

아이폰에서는 .ics 파일을 열면 캘린더로 “가져오기”가 됩니다.

아이폰에서 알림 제대로 울리게 설정
  • 캘린더 앱 → 캘린더 → 새 캘린더(예: “공모 알림”)를 만들어두면 관리가 쉬워요.
  • .ics 가져올 때 그 캘린더로 넣으면, 나중에 캘린더만 꺼/삭제하면 정리 끝.
  • 설정 → 알림 → 캘린더 → 알림 허용이 켜져 있어야 해요.

미리알림(아이폰 Reminders) 추가

웹앱에서 “미리알림”을 완전 자동으로 넣으려면 단축어(Shortcuts)를 한 번만 만들어야 해요. 일정 카드의 미리알림 버튼은 해당 단축어를 실행합니다.

단축어 만들기(처음 1번만)
  1. 아이폰 “단축어” 앱 열기 → “+” 새 단축어
  2. 이름을 정확히: 공모주 미리알림 추가
  3. 동작 추가: “미리 알림 추가”
  4. 입력(텍스트)을 제목/메모에 넣도록 설정 (앱에서 전달해줌)
  5. 저장
