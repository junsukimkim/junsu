공모 알림

가족 체크 + 캘린더(.ics) + 미리알림(단축어) + 자동 가져오기

자동 가져오기

범위: 오늘 ~ 다음달 말 (공모주 청약 일정만)

※ “증권사/균등 최소금액”은 참고 데이터로 자동 추정/매칭됩니다. 최종 확정은 반드시 공지 확인.

공모주 일정

회사명/기간/증권사/균등 최소금액이 캘린더(.ics)와 미리알림(단축어)에 함께 들어갑니다.

가족 관리

가족 목록

기본 4명 자동 생성. 필요하면 이름만 바꿔도 OK.

미리알림(단축어) 준비

아래 “미리알림” 버튼은 iOS 단축어를 실행합니다. 단축어 이름을 공모주 미리알림 추가로 만들어 주세요. (만드는 방법은 app.js 코드 아래 안내 참고)
